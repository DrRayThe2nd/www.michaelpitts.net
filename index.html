Welcome to www.michaelpitts.net!

This website is designed to convey information about its creator, Michael Pitts, to prospective employers. It runs on a Linode cloud server using Apache 2.4 on Ubuntu 22.04 LTS.

The server can only be accessed using SSH through the use of a private key. Root and password login are both disabled. The website does not make use of HTTPS due to the lack of necessity and cost associated with doing so.

For information about Michael, his experience, projects, and contact information, please visit the inline links or those at the top of the page.

Thanks for visiting!
